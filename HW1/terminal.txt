Linux terminal (GitBash) commands

1) Посмотреть где я:
   pwd

2) Создать папку
   mkdir HW1

3) Зайти в папку
   cd HW1

4) Создать 3 папки
   mkdir P1 P2 P3

5) Зайти в любоую папку
   cd P1

6) Создать 5 файлов (3 txt, 2 json)
   touch f1.txt f2.txt f3.txt f4.json f5.json

7) Создать 3 папки
   mkdir P4 P5 P6

8. Вывести список содержимого папки
   ls           - утилита для просмотра содержимого каталогов,
   ls -l (list) - форматирует вывод в виде списка с более подробной информацией,
   ls -a (all)  - включает показ скрытых файлов.

9) + Открыть любой txt файл
    vim f1.txt
10) + написать туда что-нибудь, любой текст.

11) + сохранить и выйти.
    esc and :wq

12) Выйти из папки на уровень выше
    cd ..

13) переместить любые 2 файла, которые вы создали, в любую другую папку.
    mv P1/{f1.txt,f2.txt} P2

14) скопировать любые 2 файла, которые вы создали, в любую другую папку.
    cp P1/{f4.json,f5.json} P2 

15) Найти файл по имени
    find . -type f -name f3.txt

    find   - поиск в файловой системе, файлах и папках.
    .      - текущая директория (каталог в котором будем искать)
    -name  - поиск файлов по имени.
    f3.txt - непосредственно значение по которому будем отбирать файл
    

16) просмотреть содержимое в реальном времени (команда grep) изучите как она работает.
    tail -f file_1.txt| grep 

     tail -f             - позволяет вам в реальном времени просматривать содержимое файла, обновляя вывод при появлении новых строк в файле.
     grep -r "123"       - рекурсивный поиск по всем подпапкам
     grep -l "123" *     - во всех файловых сопоставлениях в текущей директории и вывод только имен файлов.
     grep - ищет текст по шаблону

17) вывести несколько первых строк из текстового файла
    head f1.txt
    head -n 4 f1.txt - выведет на экран первые 4 строки файла

    head — с указанием имени файла, но без опций. Будут выведены на экран первые 10 строк

18) вывести несколько последних строк из текстового файла
    tail f1.txt 
    tail -n 5 f1.txt - выводить указанное количество строк из конца файла

    tail -  выводить 10 строк из конца файла

19) просмотреть содержимое длинного файла (команда less) изучите как она работает.
    less f6.txt

    less - предназначена для просмотра содержимого длинных файлов в терминале

20) вывести дату и время
    date
